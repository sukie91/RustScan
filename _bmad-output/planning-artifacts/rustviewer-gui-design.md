# RustViewer GUI 界面设计文档

**项目：** RustScan - RustViewer 3D 可视化工具
**创建日期：** 2026-03-06
**设计目标：** 为 SLAM 重建结果提供交互式 3D 可视化界面

### 2.2 控制面板详细设计

#### 2.2.1 文件加载区域
- **位置：** 面板顶部
- **高度：** 约 120px
- **组件：**
  - 主按钮："📂 打开 SLAM 结果目录..."
    - 样式：蓝色主按钮，圆角 4px
    - 尺寸：240×36px
    - 点击后打开文件夹选择对话框
  - 当前目录显示
    - 字体：等宽字体，12px
    - 颜色：灰色 (#666)
    - 显示省略路径（如果太长）
  - 文件状态列表
    - ✓ 绿色勾号：文件已加载
    - ✗ 红色叉号：文件缺失
    - ⚠ 黄色警告：文件格式错误

#### 2.2.2 图层显示区域
- **位置：** 文件加载区域下方
- **高度：** 约 140px
- **组件：**
  - 4 个复选框，每个一行
  - 复选框样式：
    - 选中：蓝色勾选框
    - 未选中：灰色边框
  - 图层名称 + 颜色指示器
    - 相机轨迹：蓝色圆点 🔵
    - 稀疏点云：蓝色圆点 🔵
    - 高斯点云：橙色圆点 🟠
    - 网格模型：灰色圆点 ⚪
  - 每个图层右侧显示数量（如果已加载）

#### 2.2.3 渲染设置区域
- **位置：** 图层显示区域下方
- **高度：** 约 160px
- **组件：**
  - 网格模式单选按钮组
    - 3 个选项：实体 / 线框 / 实体+线框
    - 默认选中：线框
  - 点大小滑块
    - 范围：1.0 - 10.0
    - 默认：5.0
    - 实时预览
  - 线宽度滑块
    - 范围：1.0 - 5.0
    - 默认：2.0
    - 实时预览

#### 2.2.4 统计信息区域
- **位置：** 渲染设置区域下方
- **高度：** 约 120px
- **组件：**
  - 5 行统计数据
  - 格式：标签 + 数值
  - 字体：等宽字体，11px
  - 颜色：深灰色 (#333)
  - 数值右对齐

#### 2.2.5 相机控制说明区域
- **位置：** 面板底部
- **高度：** 约 140px
- **组件：**
  - 6 行操作说明
  - 图标 + 操作 + 说明
  - 字体：11px
  - 颜色：中灰色 (#555)
  - 背景：浅灰色 (#f5f5f5)

## 3. 3D 视口区域设计

### 3.1 视口布局

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│                                                         │
│                    3D 场景渲染区域                       │
│                                                         │
│                  (wgpu 渲染的 3D 内容)                   │
│                                                         │
│                                                         │
│                                                         │
│                                                         │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  坐标轴指示器 (右下角)                           │   │
│  │     Y↑                                          │   │
│  │     │                                           │   │
│  │     └──→ X                                      │   │
│  │    ↙                                            │   │
│  │   Z                                             │   │

```

### 3.2 视口元素

#### 3.2.1 3D 场景内容
- **背景色：** 深灰色 (#2b2b2b) 或渐变（上深下浅）
- **网格地面：** 可选显示，浅灰色网格线
- **渲染内容：**
  1. 相机轨迹：蓝色折线，线宽 2px
  2. 稀疏点云：蓝色小点，点大小 3px
  3. 高斯点云：橙色点，点大小可调
  4. 网格模型：灰色，根据模式显示实体/线框

#### 3.2.2 坐标轴指示器
- **位置：** 右下角，距边缘 20px
- **尺寸：** 80×80px
- **样式：**
  - X 轴：红色箭头
  - Y 轴：绿色箭头
  - Z 轴：蓝色箭头
  - 箭头长度：60px
  - 标签：X, Y, Z（白色文字）

#### 3.2.3 鼠标交互反馈
- **旋转模式（左键拖拽）：**
  - 光标：抓手图标 🖐️
  - 无额外 UI 提示

- **平移模式（右键拖拽）：**
  - 光标：移动图标 ✋
  - 无额外 UI 提示

- **缩放模式（滚轮）：**
  - 无光标变化
  - 可选：短暂显示缩放比例提示

## 5. 状态栏设计

### 5.1 状态栏布局

```

│ FPS: 60 │ 相机: (0.0, 5.0, 10.0) │ 点数: 12,345 │ 内存: 245MB │

```

### 5.2 状态栏信息项

**从左到右：**
1. **FPS 计数器**
   - 格式：`FPS: XX`
   - 颜色：绿色 (>50), 黄色 (30-50), 红色 (<30)

2. **相机位置**
   - 格式：`相机: (X, Y, Z)`
   - 精度：小数点后 1 位

3. **点数统计**
   - 格式：`点数: XX,XXX`
   - 显示当前可见的点数量

4. **内存使用**
   - 格式：`内存: XXXMB`
   - 显示 GPU 内存使用量

**样式：**
- 字体：等宽字体，11px
- 颜色：浅灰色 (#999)
- 背景：深灰色 (#1e1e1e)
- 分隔符：竖线 `│`

## 7. 交互设计

### 7.1 鼠标操作

| 操作 | 功能 | 视觉反馈 |
|------|------|----------|
| 左键拖拽 | 旋转相机（绕目标点） | 光标变为抓手 🖐️ |
| 右键拖拽 | 平移场景 | 光标变为移动 ✋ |
| 滚轮向上 | 拉近相机 | 短暂显示缩放比例 |
| 滚轮向下 | 拉远相机 | 短暂显示缩放比例 |
| 双击 | 聚焦到点击位置 | 相机平滑移动动画 |

### 7.2 键盘快捷键

| 按键 | 功能 | 说明 |
|------|------|------|
| R | 重置相机 | 恢复到初始视角 |
| F | 适应场景 | 自动调整相机以显示所有内容 |
| 1 | 切换相机轨迹显示 | 开/关 |
| 2 | 切换稀疏点云显示 | 开/关 |
| 3 | 切换高斯点云显示 | 开/关 |
| 4 | 切换网格模型显示 | 开/关 |
| W | 线框模式 | 切换到线框渲染 |
| S | 实体模式 | 切换到实体渲染 |
| G | 切换网格地面 | 显示/隐藏参考网格 |
| Ctrl+O | 打开文件夹 | 弹出文件选择对话框 |
| Esc | 退出全屏 | 如果在全屏模式 |

### 7.3 拖放操作

- **拖拽文件夹到窗口：** 自动加载该文件夹中的 SLAM 结果
- **视觉反馈：** 拖拽时显示虚线边框和提示文字
- **验证：** 检查文件夹是否包含必需的文件

## 9. 动画和过渡

### 9.1 相机动画

- **重置相机：** 1 秒平滑过渡（ease-in-out）
- **适应场景：** 1.5 秒平滑过渡
- **双击聚焦：** 0.8 秒平滑过渡

### 9.2 UI 动画

- **面板折叠/展开：** 0.3 秒滑动动画
- **图层开关：** 0.2 秒淡入/淡出
- **按钮悬停：** 0.15 秒颜色过渡
- **加载进度条：** 平滑填充动画

### 9.3 性能考虑

- 所有动画使用 GPU 加速
- 3D 渲染保持 60 FPS
- UI 更新频率：30 FPS（足够流畅）

## 10. 辅助功能（Accessibility）

### 10.1 键盘导航

- 所有控件支持 Tab 键导航
- 焦点状态清晰可见（蓝色边框）
- 支持 Enter/Space 激活按钮

### 10.2 屏幕阅读器支持

- 所有按钮和控件有 ARIA 标签
- 状态变化有语音提示
- 图标有文字说明

### 10.3 对比度

- 文字与背景对比度 ≥ 4.5:1
- 重要元素对比度 ≥ 7:1
- 支持高对比度模式
