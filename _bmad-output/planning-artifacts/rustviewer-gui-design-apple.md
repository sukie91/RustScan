# RustViewer GUI 界面设计文档 - Apple 风格

**项目：** RustScan - RustViewer 3D 可视化工具
**创建日期：** 2026-03-06
**设计风格：** Apple macOS 设计规范
**设计目标：** 为 SLAM 重建结果提供优雅、直观的 3D 可视化界面

---

## 1. 设计理念

### 1.1 Apple 设计原则
- **清晰性（Clarity）**：界面元素清晰可辨，功能一目了然
- **遵从性（Deference）**：内容优先，界面元素不喧宾夺主
- **深度（Depth）**：通过层次、阴影和模糊创造视觉深度

### 1.2 视觉特征
- 浅色模式为主，深色模式可选
- 毛玻璃效果（半透明模糊背景）
- 柔和的阴影和圆角
- 精致的动画和过渡
- 充足的留白和呼吸感

## 10. 交互设计

### 10.1 按钮状态
- **默认：** 蓝色渐变 (#007AFF → #0051D5)
- **悬停：** 亮度增加 10%
- **按下：** 缩放 0.98
- **禁用：** 灰色 (#D2D2D7), 透明度 50%

### 10.2 动画
- **过渡时间：** 0.3s ease-in-out
- **弹簧动画：** 用于按钮按下
- **淡入淡出：** 0.2s

### 10.3 鼠标交互
- **悬停：** 背景色变化 + 轻微阴影增强
- **点击：** 轻微缩放反馈
- **拖拽：** 光标变化 + 视觉反馈

## 14. 实现建议

### 14.1 毛玻璃效果
- 使用 backdrop-filter: blur(20px)
- 半透明背景 rgba(246, 246, 246, 0.8)

### 14.2 阴影
- 使用多层阴影创造深度
- 避免过重的阴影

### 14.3 动画
- 使用 ease-in-out 缓动
- 保持 60fps 流畅度
