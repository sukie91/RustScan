# Chunked Training Handoff Template

Use this note when a chunked-training story is still `in_progress` or blocked at the end of a work session.

```md
# Handoff Note

Story: <epic-story-id and title>
Status: in_progress | blocked
Date: <YYYY-MM-DD>

## Current State

<1-3 sentences summarizing what is already implemented>

## Files Changed

- <path>
- <path>

## Last Verified Step

- Command: `<command>`
- Result: <passed | failed | not run>

## Next Step

<the exact next action the next developer should take>

## Known Risk or Blocker

<specific blocker, uncertainty, or edge case>
```

Store the filled note next to other implementation artifacts and include the story id in the filename when needed.
